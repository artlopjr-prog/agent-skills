# Web Design Guidelines

100+ rules: accessibility, focus, forms, animation, typography, images, performance, navigation, dark mode, touch, i18n.

## Key Rules
- Semantic HTML, alt text, aria-labels, contrast ratios
- focus-visible, keyboard navigation, modal focus traps
- label/input association, autocomplete attributes
- prefers-reduced-motion, compositor-only animations (transform/opacity)
- tabular-nums, curly quotes, min 16px body
- width/height on images, loading=lazy, WebP/AVIF
- Virtualize lists >100 items, preconnect hints
- URL reflects state, deep links work
- prefers-color-scheme, CSS custom properties
- 44x44px tap targets, touch-action: manipulation
- Intl.DateTimeFormat, Intl.NumberFormat, lang attribute
